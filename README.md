# ExamenUnidad3_Patrones_Gonzalez_Vazquez_Melanie_Estefania

Este examen emplea los patrones de diseño estructurales: Decorador y Fachada
Es un gestor de audio con efectos sonoros opcionales, donde en el Decorador, se emplea para agregar efectos a los audios simples que el usuario carga; mientras que Fachada simplifica los subsistemas, creando facilidad al seleccionar cada efecto en vez de c estar creando objetos en el Main.

Es una simulación de consola en como funciona un gestor de audio (solo en string). El usuario selecciona un archivo, escoge si ponerle efectos o no, en caso de que si, aparecerá varios efectos que podra agregarle al audio, modificandolo a su gusto, una vez satisfecho con su creación, se reproducirá el audio junto con los efectos.
